# My MCP Servers

Welcome to the `my_mcp_servers` repository. This repository will contain a collection of various Model Context Protocol (MCP) servers.

## Available Servers

### 1. Agno MCP Server

The first MCP server available in this repository is for **Agno**.

**Description**: 
The Agno MCP Server provides tools to interact directly with the Agno documentation. It allows AI agents to search for topics, get relevant documentation links, and read the full content of specific pages from the Agno documentation. Here it is the [repository link](https://github.com/alessiofontanarossa/my_mcp_servers/tree/main/agno_mcp_server).
